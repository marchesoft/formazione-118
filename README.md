# Sistema di Gestione Corsi Aziendali

Questa è un'applicazione web semplice per la gestione dei corsi di formazione del personale.

## Funzionalità
- Visualizzazione lista corsi.
- Creazione e modifica corsi.
- Gestione partecipanti.
- Sistema di chat integrato per ogni corso.
- Filtro ricerca per data e location.

## Tecnologie Utilizzate
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- Supabase per la persistenza dei dati e gestione utenti.

## Come eseguire localmente
Basta aprire il file `index.html` in qualsiasi browser moderno.

## Pubblicazione su Vercel

Questo progetto è ottimizzato per Vercel. Per distribuirlo:
1. Carica il codice su GitHub.
2. Vai su [Vercel](https://vercel.com).
3. Importa la repository `corsi-formazione`.
4. Vercel rileverà automaticamente le impostazioni e distribuirà l'app.

Le impostazioni in `vercel.json` garantiscono il corretto funzionamento delle rotte e degli URL puliti.
